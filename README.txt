Como primer paso una vez clonado este repositorio ejecutar el comando "npm i" para instalar todas la dependencias base de este repositorio.

Atención: inicialmente el archivo dbConnection.js estará comentado hasta tanto se coloque un MONGO_URI y un DB_NAME en el archivo .env, de lo contrario dará un error al levantar el servidor.
Por el mismo motivo también se encuentra momentáneamente comentada la importación de dicho archivo en app.js y su utilización en el server del mismo archivo.

En la carpeta docs es donde van a estar los archivos .yaml con la documentación de la API.

En la carpeta test es donde se van a utilizar las dependencias chai, mocha y supertest.

Completar el archivo README.md
